# Mangatools

**Mangatools** — это набор инструментов для автоматизации скачивания, сортировки и подготовки манги в виде картинок. Идеально подходит для скачивания манги с сайта [manga-chan.me](https://manga-chan.me) и подготовки готовой коллекции для чтения.

---

## Возможности

1. **Скачивание манги**
   - Поддерживается скачивание всех глав по ссылке на страницу манги.
   - Проверяет, есть ли файл в папке, чтобы не скачивать повторно.
   - Отображает прогресс загрузки через красивый прогресс-бар.

2. **Парсинг названий глав**
   - Автоматически извлекает номера томов и глав с сайта.
   - Сохраняет данные в JSON для корректного именования папок и файлов.

3. **Разархивация и сортировка**
   - Распаковывает ZIP-архивы с мангой.
   - Создает структуру папок `Berserk - Том X / Y. Название главы`.
   - Поддержка будущей интеграции с RAR и 7z.

4. **Очистка ненужных файлов и конвертация AVIF**
   - Удаляет временные и лишние файлы (список редактируется через `files_to_delete.txt`).
   - Конвертирует изображения `.avif` в `.png` без потерь качества.

5. **Прогресс-бар**
   - Все этапы отображаются через Rich, удобно отслеживать прогресс работы.
