# Mangatools

**Mangatools** — это набор инструментов для автоматизации скачивания, сортировки и подготовки манги в виде картинок. Проект оптимизирован для работы с сайтом [manga-chan.me](https://manga-chan.me) и позволяет подготовить мангу к чтению в готовом виде.

---

## Возможности

1. **Скачивание манги**
   - Скачивает все главы манги по ссылке на страницу.
   - Проверяет наличие файлов в папке, чтобы избежать повторных загрузок.
   - Отображает прогресс загрузки через красивый прогресс-бар Rich.

2. **Парсинг названий глав**
   - Автоматически извлекает номера томов и глав с сайта.
   - Сохраняет информацию в JSON для корректного именования папок и файлов.

3. **Разархивация и сортировка**
   - Распаковывает ZIP-архивы с мангой.
   - Создаёт структуру папок вида `Berserk - Том X / Y. Название главы`.
   - Поддержка будущей интеграции с RAR и 7z.

4. **Очистка ненужных файлов и конвертация AVIF**
   - Удаляет временные и лишние файлы (список редактируется через `files_to_delete.txt`).
   - Конвертирует изображения `.avif` в `.png` без потерь качества, кроссплатформенно через `imageio[avif]`.

5. **Прогресс-бар**
   - Все этапы отображаются через Rich, удобно отслеживать прогресс работы на каждом шаге.
